# Demand Elasticity Analysis using Python

#  Objective
This project analyzes the relationship between price and quantity demanded across different product categories (FMCG, luxury, and normal goods) to understand consumer responsiveness and validate the law of demand.

# Tools & Technologies
- Python (Pandas, NumPy)
- Matplotlib / Seaborn
- Jupyter Notebook


# Key Analysis Performed
- Data cleaning and preprocessing
- Handling missing values and outliers
- Price elasticity calculation
- Category-wise elasticity comparison
- Visualization using boxplots and log-log scatter plots


# Key Insights
- FMCG goods exhibit low elasticity (inelastic demand)
- Luxury goods show higher elasticity (price-sensitive demand)
- A small portion of observations reflect Veblen-type behavior
- Overall, an inverse relationship between price and demand is observed
